# gweibo
获取山人最新微博

## 运行方式
`./gweibo  2>>err | tee posts.log`

## 目标
1. 全自动化获取山人微博
2. 有恰当的预警机制
3. 有良好的保存机制(支持文字、图片；历史微博的归档)
4. 只专注获取山人最新微博

## 已完成 
1. 简单获取山人最新微博并输出到标准输出
2. cookie可配置
3. 简单的配置热更新（无错误处理）
